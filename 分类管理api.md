## 添加分类
- 接口地址：http://127.0.0.1:8080/category/addType
- 返回格式：json
- 请求方式：get
- 请求示例：http://127.0.0.1:8080/category/addType?Kinds=
- 接口备注：
### 请求参数

字段 | 类型 | 必填 |说明
---|---|-----|-----
Kinds | String | Y |分类

### 无返回值
